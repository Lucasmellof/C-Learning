# Printf

| Char   | Printed as                     |
|--------|--------------------------------|
| %c     | Char                           |
| %s     | String                         |
| %d     | Signed int                     |
| %e, %E | Scientific float notation      | 
| %f     | Float                          |
| %hi    | short (signed)                 | 
| %hu    | short (unsigned)               | 
| %Lf    | long double                    |
| %n     | prints empty                   |
| %d     | Decimal int (base 10)          | 
| %i     | Decimal int (auto-detect base) | 
| %o     | octal (base 8) int             | 
| %x     | hexadecimal (base 16) int      | 
| %p     | pointer                        | 
| %u     | unsigned decimal               | 
| %%     | Will print '%' symbol          |
